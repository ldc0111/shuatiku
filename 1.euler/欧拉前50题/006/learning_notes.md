#### 题目大意

## **平方的和与和的平方之差**

前十个自然数的平方的和是

12 + 22 + … + 102 = 385

前十个自然数的和的平方是

(1 + 2 + … + 10)2 = 552 = 3025

因此前十个自然数的平方的和与和的平方之差是 3025 − 385 = 2640。

求前一百个自然数的平方的和与和的平方之差。



#### 基本思路

1，直接模拟



2，数学公式

(1,,,,,,100) = n(n + 1) / 2;



n^3 - (n - 1)^3 = 3n^2 - 3n + 1

,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,,

2^3 - 1  = 3 * 2 ^2 - 3 * 2 + 1;



n^3 - 1^3 = 3（n^2,,,,,,,,,1） - 3(n +(n -1)  + 1) + n - 1;

n^3 =   3（n^2,,,,,,,,,1） -  3(n +(n -1)  + 1) + n;

（n^2,,,,,,,,,1） = (n^3  + 3(n +(n -1)  + 1) - n)/3;

n^2,,,,,,,,,1） = (n^3  + 3*n(n - 1)/2 - n)/3;
